Movile First

Mobile First es una filosofía o estrategia que se basa en una premisa clara: el móvil tiene que ser el primer dispositivo que se a de tener en cuenta a la hora de diseñar una página web. Es decir, hay que priorizar aquellos contenidos y/o funcionalidades que son imprescindibles para el usuario y desechar todo lo demás. Una vez esto, es cuando se empieza a pensar en la optimización para el resto de resoluciones mayores, como tablets y ordenadores.
